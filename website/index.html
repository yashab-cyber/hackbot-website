Intelligence Modules

Built-in security intelligence beyond simple scanning.

🛡️ CVE / Exploit Lookup

Real-time vulnerability intelligence powered by the NVD and GitHub exploit search.

  • NVD Lookup — full details (CVSS, severity, CWEs)
  • Keyword search with severity filtering
  • Nmap→CVE automatic mapping
  • Exploit PoC search on GitHub

🌐 OSINT Module

Passive intelligence gathering for reconnaissance — no active scanning required.

  • Subdomain enumeration (crt.sh + DNS brute)
  • DNS records (A, AAAA, MX, NS, TXT, CNAME, SOA)
  • WHOIS / RDAP lookup
  • Email harvesting & tech stack fingerprinting
/osint example.com

🗺️ Network Topology

Parse nmap or masscan output into interactive network topology maps.

  • Nmap text & XML parsing
  • Interactive D3.js force-directed graph
  • ASCII rendering in terminal
  • Subnet grouping & markdown summary
/topology scan_results.txt

Installation

Get started in seconds. Works on Linux, macOS, and Windows.

Linux / macOS
curl -sSL https://raw.githubusercontent.com/yashab-cyber/hackbot/main/install.sh | bash

Full install with security tools:

Full install
curl -sSL https://raw.githubusercontent.com/yashab-cyber/hackbot/main/install.sh | bash -s full
Full install (GUI + PDF + all)
pip install "hackbot[all] @ git+https://github.com/yashab-cyber/hackbot.git"
Minimal (CLI only)
pip install git+https://github.com/yashab-cyber/hackbot.git
Isolated install
pipx install "hackbot[all] @ git+https://github.com/yashab-cyber/hackbot.git"
From source (development)
git clone https://github.com/yashab-cyber/hackbot.git
cd hackbot
pip install -e ".[all,dev]"
Docker
docker build -t hackbot .
docker run -it -e HACKBOT_API_KEY=your-key --network host hackbot
Windows (PowerShell)
pip install "hackbot[all] @ git+https://github.com/yashab-cyber/hackbot.git"

Or clone and run the installer:

Installer
git clone https://github.com/yashab-cyber/hackbot.git
cd hackbot
install.bat

Architecture

Clean, modular Python architecture designed for extensibility.

project structure 
hackbot/
├── cli.py               # Main CLI — 40+ commands, interactive REPL
├── config.py            # Configuration management (YAML + env vars)
├── memory.py            # Session memory & conversation summarization
├── reporting.py         # Report generation (HTML / MD / JSON)
├── core/
│   ├── engine.py        # AI engine — 10 providers, LLM communication
│   ├── runner.py        # Tool execution (sandboxed subprocess)
│   ├── cve.py           # CVE / exploit lookup (NVD + GitHub)
│   ├── osint.py         # OSINT recon (subdomains, DNS, WHOIS)
│   ├── topology.py      # Network topology parser & visualizer
│   ├── compliance.py    # Compliance mapping (4 frameworks)
│   ├── campaigns.py     # Multi-target campaign system
│   ├── diff_report.py   # Assessment diff engine
│   ├── pdf_report.py    # Professional PDF reports
│   ├── plugins.py       # Custom plugin system
│   ├── remediation.py   # AI Remediation engine
│   └── proxy.py         # HTTP Proxy / Traffic Capture
├── modes/
│   ├── chat.py          # Chat mode (Q&A + auto-save)
│   ├── agent.py         # Agent mode (autonomous testing)
│   └── plan.py          # Planning mode (8 templates)
├── gui/
│   ├── app.py           # Flask backend (25+ API routes, SSE)
│   └── templates/
│       └── index.html   # Single-page frontend (dark theme, D3.js)
└── ui/
    └── __init__.py      # Rich terminal UI components
